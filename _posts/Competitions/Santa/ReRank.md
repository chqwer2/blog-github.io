### Cost

**reconfiguration cost**  

each step is the square root of the number of links rotated at that step, 

```
return np.sqrt(diffs.sum())
```

**Color cost**

the sum of the absolute change in the color components from the previous step's position to the current position

```
return np.abs(image[to_position] - image[from_position]).sum() * color_scale
```
